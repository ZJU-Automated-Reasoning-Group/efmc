Welcome to EFMC & EFSMT Web UI

A web interface for SMT-based Software Model Checking tools.

This web interface allows you to interact with EFMC and EFSMT tools through your browser.

EFMC

EFMC is a toolkit for verifying program properties using SMT-based verification engines. It implements multiple approaches, such as template-based verification, property-directed reachability (PDR), Houdini, and k-induction.

Try EFMC
EFSMT

EFSMT is a tool for solving "exists-forall" SMT problems. It provides a powerful interface for handling complex SMT formulas with nested quantifiers.

Try EFSMT
